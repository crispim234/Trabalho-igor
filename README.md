# React + TypeScript + Vite

```Para rodar o Projeto React+Vite+Typescript
    1) Pré-requisitos
• Node.js instalado (recomendo LTS — v18+).
• VS Code instalado.
• Familiaridade básica com terminal (cmd/PowerShell/macOS Terminal).
Verifique a versão do NodeJS e NPM em sua máquina com os seguintes comandos no
terminal:
• node -v
• npm -v
    2) Criar o projeto com Vite (npm)
No terminal (escolha uma pasta onde quer o projeto):
Usando npm:
• npm create vite@latest meu-app -- --template react-ts
• cd meu-app
• npm install
    3) Abrir no VS Code
No terminal já na pasta do projeto:
• code .
    4) Rodar o servidor de desenvolvimento
No terminal do projeto:
• npm run dev
```
